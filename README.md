fy 是一个命令行英文词典工具，支持查询单词释义、同义词辨析等功能。
示例：
- 查询单词：`$ fy insist`
- 查询单词及其同义词辨析：`$ fy -d insist`
- 查看帮助：`$ fy -h`

```
Usage:
  fy [text] [flags]

Flags:
  -d, --discriminate   Discriminate words
  -h, --help           help for fy
  -q, --query string   Query words
  -v, --verbose        Verbose mode
```
